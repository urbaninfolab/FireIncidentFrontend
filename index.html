Active Fire Incidents Map Information

About

This page provides live weather information of active fire incidents as an interactive map. It also provides predicted smoke fallout from each fire, as well as live air quality data. We currently collect and showcase past and present fires from over 20+ cities.

Credits

It was created underneath the Smart Cities Project, under the Good Systems Initiative in collaboration with The University of Texas at Austin.

Additional support for this project comes from the Texas Advanced Computing Center, The Georgia Forestry Commission, and the Austin Fire Department.

Welcome to Active Fire Incidents Map

Guide

Fire Icon (Active Fire): The red fire icon represents active fires - fires that are currently burning and require immediate attention. These are ongoing fire incidents that fire departments are actively responding to.
Deactive Fire Icon (Inactive Fire): The gray deactive fire icon represents inactive fires - fires that have been extinguished or are no longer active. These are historical fire incidents that have been resolved. You can toggle the display of inactive fires using the "Inactive Fire" checkbox in the data filter menu.
Layer Control: Click the layer icon to open the data filter menu. Select different data layers including Fire Data (Active/Inactive Fires), Smoke Forecasts, Air Quality Data, Shapefiles (Fire Risk, AFD Coverage), Points of Interest, and Urban Mobility data.
Statistics: Click the stats icon to view detailed statistics and analytics about the fire incidents displayed on the map, including fire counts, trends, and other relevant data.
Today
Date Selection: Use the date dropdown menu to filter fire incidents by time period. Options include Today, Yesterday, Past 3 Days, or Custom date range. You can also use the date picker to select a specific date.
